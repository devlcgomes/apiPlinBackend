## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

 - A variação de dados dependendo do horário do dia foi algo interessante de fazer.

## Descreva a funcionalidade e o por quê da utilização das bibliotecas
escolhidas por você para concluir o desafio.
 - A api trás produtos por meio do cardápio. Contem 2, um para modo diurno e um para noturno, os produtos podem ser organizado por categorias.
 - Possui um CRUD para cardapio, categorias e produtos.
 - A API conta com documentação via SWAGGER.
